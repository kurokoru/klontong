## Klontong

Klontong is groceries market in Indonesia slang. This place for my support to sell my used goods.

## Stack
- JS
- Bun
- Monorepo: Turborepo
- Backend: Hono
- Frontend: React
- Validation: Zod
- Database: Postgresql
- ORM: Prisma
- Container: Docker
